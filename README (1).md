# MANMOH — Netflix Inspired Streaming Platform

A Netflix-inspired streaming platform homepage built with pure HTML and CSS. No frameworks, no libraries — just clean front-end code from scratch.

## 🔗 Live Demo
👉 [https://owss-maroof.github.io/MANMOH-netflix-clone](https://owss-maroof.github.io/MANMOH-netflix-clone)

---


## ✨ Features

**Header**
- Custom MANMOH logo in Netflix-style red
- Language selector (English, Hindi, Spanish, French)
- Sign In button with hover effect
- Fully responsive on all screen sizes

**Hero Section**
- Full viewport background image with dark overlay
- Bold headline and subheadline
- Email input with Get Started CTA button
- On mobile — input and button stack vertically

**Movie Row**
- Horizontally scrollable movie cards
- Smooth zoom and brightness effect on hover
- Mini scrollbar for clean look
- Cards resize on tablet and mobile

**Contact Section**
- Name, email, and message input fields
- Send Message button in red
- Contact info with emoji icons

## 📱 Responsive Breakpoints

| Screen Size | Behaviour |
|---|---|
| Desktop (>768px) | Full layout, all elements visible |
| Tablet (≤768px) | Smaller header, smaller text, smaller cards |
| Mobile (≤480px) | Language selector hidden, CTA stacks vertically, cards shrink |

## ⚙️ How to Run Locally

No server needed — just open in browser:

```bash
git clone https://github.com/owss-maroof/MANMOH-netflix-clone.git
cd MANMOH-netflix-clone
# open index.html in any browser
```
